Para poder utilizar el ejemplo debera:

1) crear un nuevo proyecto Laravel
	:~$ laravel new clase3-ej2
	
2) copiar el contenido de esta carpeta completo a la carpeta del nuevo 	proyecto recien creado. 
	
3) Al pegar las carpetas y archivos solicitara sobreescribir los mismo, lo que se confirmara.

4) Generar la key
    php artisan key:generate

5) crear en MySQL o MariaDB una base de datos llamada ejcrud

6) Ejecutar las migraciones en la carpeta del proyecto
    php artisan migrate
